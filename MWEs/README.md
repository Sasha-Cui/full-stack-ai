# MWEs

This directory holds the core learning materials for the repository.

## CPU-Safe Starting Points

- `pytorch/`
- `Scaling_Laws/`
- `agentic_rl_workshop.ipynb`

These are the best places to begin if you want runnable material without GPUs, containers, or live API keys.

## LLM / Agent Application Material

## Robotics / Embodied AI
