# full-stack-ai
Fall 2025 Working Group on Becoming Full-Stack AI Researchers at Yale University.  This working group consists of graduate students from Computer Science, Statistics and Data Science, Electrical and Computer Engineering, Computational Biology, Applied Physics, Applied Mathematicis, and other departments.  We cover different aspects of end-to-end AI development and engineering.

The course website is https://sashacui.com/full-stack.html
